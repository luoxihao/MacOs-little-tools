# 安装指南

## 1. 依赖安装

双击执行shell脚本 app，会要求打开一个文件，在这里选中 `workflow` **jpeg质量压缩**文件夹内的 `install_brew_imagemagick.sh` 文件。
此时会跳出一个程序终端。

 不要慌张！！！依照程序终端的提示操作就好，全部是中文的。

当显示如下时安装完成：
```
Homebrew 已经安装。
jpegoptim 已经安装。
验证安装...
jpegoptim 已成功安装。
```


此时依赖安装部分已经完成！！！

## 2. Workflow 安装

双击扩展名为 `.workflow` 的文件后，会弹窗“快速操作”安装器，点击安装即可。
